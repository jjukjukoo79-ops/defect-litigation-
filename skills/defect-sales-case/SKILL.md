# 하자소송 영업·사건관리 스킬

"기본자료 베이스(①~④)"를 **돈이 되는 사건**으로 연결하는 영업·관리 엔진.
사무실 단지 자산(`N:\# 아파트별 관련자료\`)을 잠재고객 풀로 활용한다.
변호사법 안전선: 사건 진행·일정·자료 관리까지. 수임 여부·법률판단·광고문구는 변호사 승인.

---

## 1. 영업의 핵심 = 담보책임기간 '만료 임박' 단지 ★

하자 손해배상은 **담보책임기간 내에 제소**해야 하므로, **만료가 다가오는 단지일수록
영업 골든타임**이다. 사용승인일에서 역산해 우선순위를 매긴다.

| 잔여기간 | 우선순위 | 의미 |
|----------|----------|------|
| **만료 6개월 이내** | 🔴 최우선 | 지금 잡지 않으면 권리 소멸 위험 — 즉시 컨택 |
| 만료 6~18개월 | 🟠 우선 | 조사·감정 준비기간 확보 가능 |
| 만료 18개월~ | 🟡 일반 | 관계 구축·예비 접촉 |
| 주요 분기점 | — | 사용승인 **+2년/+3년/+5년/+10년** 도래 단지 집중 |

> 입력: 단지명·세대수·**사용승인일** → 출력: 공종별 잔여기간 + 우선순위 색.

## 2. 잠재단지 발굴 → 수임 파이프라인

```
[발굴] 사용승인 N년 경과 단지 추출
   → [접촉] 입주자대표회의·관리사무소 컨택
   → [설명회] 하자진단·소송 설명(제안서 활용)
   → [예비조사] 육안 하자조사·하자판정(①스킬)
   → [수임] 총회 결의·소송위임 계약·위임장 징구
   → [착수] 사건 등록 → 사건관리(아래 3)
```

- **제안서·설명자료**: 자료 베이스의 `하자소송제안서.pptx`, `하자소송 의견서.hwp` 활용.
- **입대의 의사결정**: 공용부분 소송은 보통 **입주자 총회 결의 + 채권양도**가 필요 → 일정 관리.

## 5. 기일·비용 관리

- **기일관리**: 변론기일·감정 현장조사일·총회일·**제소기한(담보기간 만료일)**을 캘린더화.
  특히 만료일은 🔴로 별도 알림.
- **비용관리**: 감정료 예납액·인지/송달료·실비, 단지별 정산. 성공보수 조건 기록.

## 6. 소제기 실익 판단 ★ (수임 전 필수 — 위임계약/의견서 기준)

수임 전 **"받을 돈 − 들 돈 − 성과보수"**를 먼저 계산한다. 실익 없는 단지는 솔직히 말리는 것도 전문성.
**실익 기준 = 세대당 판결금 200만원.**

```
총 보수비(감정 추정)
 × 채권양도율
 × (1 − 책임제한율: 사용검사 1년경과당 ≈5%, 통상 20~40%)
 = 예상 판결 원금
 ÷ 양도세대수 = 세대당 판결금   ← 200만원 이상이면 실익 있음
 − 로펌 대납비용 − 성과보수 = 아파트 실수령
```

**위임계약 비용구조(제8조)** — 로펌이 아래를 **대납** 후 정산(전부 패소 시 미청구):
- 하자진단비(≈2,000만) · 법원감정료(세대당 ≈10만) · 인지대 · 송달료 · 파취조사비 · 기술자문료.

**성과보수(제9조)** — 종결단계·세대당 판결금 구간별:
| 종결단계 | 율 |
|----------|----|
| 소제기 전 합의 | 1% |
| 감정물 납부 전 종결 | 2% |
| 감정 후 판결 (세대당 200만 미만 / 200~250만 / 250만↑) | 10% / 12% / 12.5% |

- 최초 소장 청구액은 보통 **세대당 200만원 내외**로 책정(감정 후 청구취지변경).
- 표준 산출은 `소제기_실익_판단서_양식` 템플릿 또는 **허브의 실익 계산기** 사용.
