# 소제기 실익 판단서 (○○아파트)

> 수임·소제기 전 "받을 돈 − 들 돈"을 미리 계산하는 내부 검토표.
> 의견서(남양시티프라디움2차) 방식. 최종 의견은 변호사 확정.

## 0. 단지 기초정보
| 항목 | 값 |
|------|----|
| 단지명 / 세대수 | ○○아파트 / ○○○세대 |
| 사용검사일 | 20○○. ○. |
| 경과연수(오늘 기준) | ○년 |
| 청구가능 하자(도과 후 잔존) | (예: 5년차 도과 → 10년차만) |
| 채권양도 세대수 / 비율 | ○○○세대 / ○○% |

## 1. 실익 계산
| 단계 | 산식 | 금액(원) |
|------|------|----------|
| ① 총 보수비(감정 추정) | | |
| ② × 채권양도율 | ① × ○○% | |
| ③ × (1 − 책임제한율) | ② × (1 − ○○%) | |
| ④ 예상 인용 원금 | = ③ | |
| ⑤ − 하자진단비 | 약 20,000,000 | (−) |
| ⑥ − 법원 감정료 | 세대수 × 100,000 | (−) |
| ⑦ 단지 실수령 추정 | ④ − ⑤ − ⑥ | |
| ⑧ 세대당 회수액 | ⑦ ÷ 양도세대수 | |

> 책임제한율 가늠: **사용검사 1년 경과당 ≈ 5%**(통상 20~40%). 예: 8년 경과 ≈ 40%.

## 2. 판정
- [ ] 세대당 회수액이 **충분** → 소제기 실익 있음(진행 검토)
- [ ] 세대당 회수액이 **수십만원대 등 과소** → 실익 없음 의견 가능
