# 소제기 실익 판단서 (○○아파트)

> 수임·소제기 전 "받을 돈 − 들 돈 − 성과보수"를 미리 계산하는 내부 검토표.
> 위임계약(제8·9조) 비용구조 + 의견서(남양시티프라디움2차) 방식. 최종 의견은 변호사 확정.
> **실익 기준 = 세대당 판결금 200만원.**

## 0. 단지 기초정보
| 항목 | 값 |
|------|----|
| 단지명 / 세대수 | ○○아파트 / ○○○세대 |
| 사용검사일 / 경과연수 | 20○○. ○. / ○년 |
| 청구가능 하자(도과 후 잔존) | (예: 5년차 도과 → 10년차만) |
| 채권양도 세대수 / 비율 | ○○○세대 / ○○% |
| 예상 종결단계 | 합의 / 감정 전 종결 / 감정 후 판결 |

## 1. 판결 원금 산정
| 단계 | 산식 | 금액(원) |
|------|------|----------|
| ① 총 보수비(감정 추정) | | |
| ② × 채권양도율 | ① × ○○% | |
| ③ × (1 − 책임제한율) | ② × (1 − ○○%) | |
| ④ 예상 판결 원금 | = ③ | |
| ⑤ **세대당 판결금** | ④ ÷ 양도세대수 | **(기준 200만원)** |

> 책임제한율 가늠: 사용검사 1년 경과당 ≈5%(통상 20~40%). 예: 8년 경과 ≈ 40%.

## 2. 비용·성과보수 차감 (위임계약 반영)
| 항목 | 산식 | 금액(원) |
|------|------|----------|
| ⑥ 로펌 대납비용 | 하자진단비(≈2,000만) + 법원감정료(세대수×≈10만) + 인지·송달·파취조사비·기술자문료 | (−) |
| ⑦ 성과보수 | ④ × 성과보수율 | (−) |
| ⑧ 아파트 실수령 | ④ − ⑥ − ⑦ | |

**성과보수율(제9조)**
| 종결단계 | 율 |
|----------|----|
| 소제기 전 합의 | 합의금액의 1% |
| 감정물 납부 전 종결 | 합의금액의 2% |
| 감정 후 판결 — 세대당 판결금 200만 미만 | 10% |
| 감정 후 판결 — 세대당 판결금 200만~250만 | 12% |
| 감정 후 판결 — 세대당 판결금 250만 이상 | 12.5% |

> ※ 전부 패소 시 로펌은 대납비용을 청구하지 않음(제8조 제2항).

## 3. 판정 (기준 = 세대당 판결금 200만원)
- [ ] **세대당 판결금 ≥ 200만원** → 실익 있음(진행 검토)
- [ ] **200만원 미만** → 실익 낮음/없음 의견 가능
- [ ] 아파트 실수령(⑧)이 0 이하 → 실익 없음
