# 하자소송 전문 에이전트 (Defect-litigation)

공동주택 **하자소송 전문 로펌**으로 거듭나기 위한 실무 도구 모음.
재료 베이스 = `N:\# 아파트별 관련자료\1. 영업 자료\14. 하자소송관련\`
(건설감정실무 / 하자판정기준 고시 / 분쟁조정 사례집 / 공동주택관리 매뉴얼 / 의견서·제안서 샘플)

연결·관리하는 **⑤영업·사건관리**.


## 교육자료 (education/)
- `하자소송_영업사무장_교육자료.pdf` — 영업 사무장 입문 교육용(8장).
- `하자소송_회의교육_정리(연수자료+의견서).pdf` — 변호사 연수자료 + 실전 의견서 요약 + **회의 논의 질문**.

> 근거: 하혜용 변호사 연수자료(2026.6.17) + 의견서(남양시티프라디움2차). 핵심 수치
> (책임제한 연 5%·채권양도율·보증금 시효·감정료 세대당 10만 등)는 스킬 ②④⑤에 반영됨.
